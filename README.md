# 🌐 Milan Varshney — Portfolio

My personal developer portfolio showcasing projects, skills, and experience. Built from scratch with vanilla HTML, CSS & JavaScript.

**🔗 Live:** [milan-varshney.github.io/portfolio](https://milan-varshney.github.io/portfolio/)

## ✨ Features

- Dark, techy UI with a clean minimal layout
- Interactive terminal typing animation
- Cursor-tracking glow effect
- Scroll-triggered reveal animations
- Animated stat counters
- Fully responsive — works on mobile, tablet & desktop

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript
- Google Fonts (JetBrains Mono + Inter)

## 🚀 Setup

No build tools needed. Just open `index.html` in a browser or deploy via GitHub Pages.
